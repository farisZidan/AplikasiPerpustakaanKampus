Ini adalah aplikasi pengelolaan perpustakaan yang dibuat dengan Framework Laravel 12. 
Proyek ini dibuat untuk memenuhi tugas akhir mata kuliah Pemrograman Framework Universitas Semarang.

Sytling aplikasi ini menggunakan vite dan tailwind CSS; Database menggunakan MySql; Views menggunakan Blade.

Fitur-fitur yang ada di aplikasi ini:
1. Pengelolaan Buku
2. Pengelolaan Anggota perpustakaan
3. Pengelolaan Transaksi Peminjaman Buku

Syarat Menjalankan Aplikasi ini:
1. Node.js
2. MySql Server (XAMPP, Laragon etc)
